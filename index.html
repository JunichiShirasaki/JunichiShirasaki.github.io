CC5. Accessing data: Scraper and API

The task in this code challenge is to create and embed two types of charts: using an API and web scraping.

Chart9
The chart using API: The difference in the trends of carbon dioxide emissions in Japan and the United Kingdom

This chart shows the trends in per capita carbon dioxide emissions in Japan and the United Kingdom since 1970.

Data Source: The World Bank
Base URL for API of the World Bank: https://api.worldbank.org/v2/
The endpoint of the API I used in this chart: country/JP;GB/indicator/EN.GHG.CO2.PC.CE.AR5
To collect data using the World Bank API, I identified the codes of countries for Japan and the United Kingdom, and the indicator code for Carbon Dioxide emissions.
The full URL used in the chart is as follows:
https://api.worldbank.org/v2/country/JP;GB/indicator/EN.GHG.CO2.PC.CE.AR5?format=json&per_page=1000
