My project page
How do OECD countries cluster into distinct socio-economic structures, and how much heterogeneity remains within these structural clusters?

Portfolio Page | GitHub

Project aims

Governments and media often compare countries by referring to their rank among OECD members, implicitly treating the OECD as a homogeneous group. This project challenges that assumption by examining whether OECD countries are structurally comparable. Using six socio-economic indicators, it explores similarities and differences across OECD countries and identifies clusters with similar structural profiles. The aim is to show that a cluster-based perspective provides a more informative understanding of cross-country differences.

Data & Tools

All numerical data used in this project are sourced from the World Bank’s open API. Data collection, cleaning and preparation were fully automated using Python in a single Google Colab notebook, which ensures full reproducibility.

Colab notebook: here

After retrieving country-level data from the World Bank, I harmonised country identifiers and constructed period averages (2015 - 2019) to reduce short-term volatility. Data from 2020 onwards are excluded because the COVID-19 pandemic and Russia's invasion of Ukraine may introduce unusually large shocks, making cross-country structural comparison less stable. To make indicators comparable, each figure is standardised using z-scores. In other words, for each indicator, I subtract the OECD average from a country's value and then divide the result by the OECD standard deviation, using data averaged over 2015 - 2019. These indicators form the basis for the heatmap and k-means clustering. Visualisation is implemented using Vega-Lite.
One challenge was ensuring consistent joins between statistical data and a geographic TopoJSON file. I addressed this by matching ISO numeric codes and validating joins programmatically in Python. This TopoJSON is also collected in the Google Colab notebook above.

Conclusion

This project shows that treating OECD countries as a single homogeneous group can be misleading. By combining standardised indicators with clustering and visualisation, it demonstrates that countries often labelled as comparable occupy very different structural positions. Rather than focusing on simple rankings, a more informative approach is to consider which structural group a country belongs to and how diverse that group itself is.
