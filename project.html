My project page

How do OECD countries cluster into distinct socio-economic structures, and how much heterogeneity remains within these structural clusters?

Portfolio Page | GitHub

Project aims

Governments and media often compare countries by referring to their rank among OECD members, implicitly treating the OECD as a homogeneous group. This project challenges that assumption by examining whether OECD countries are structurally comparable. Using six socio-economic indicators, it explores similarities and differences across OECD countries and identifies clusters with similar structural profiles. The aim is to demonstrate that OECD countries can be more informatively understood as belonging to several broad structural types rather than as a single homogeneous group.

Data & Tools

All numerical data used in this project are sourced from the World Bank's open API. Data collection, cleaning and preparation were automated using Python in a single Google Colab notebook, ensuring reproducibility.
After retrieving country-level data from the World Bank, I harmonised country identifiers and constructed period averages (2015 - 2019) to reduce short-term volatility. Data from 2020 onwards are excluded because the COVID-19 pandemic and Russia's invasion of Ukraine may introduce unusually large shocks, making cross-country comparison less stable. To make indicators comparable, each figure is standardised using z-scores. In other words, for each indicator, I subtract the OECD average from a country's value and then divide the result by the OECD standard deviation, using data averaged over 2015 - 2019. These indicators form the basis for the heatmap and clustering. Visualisation is implemented using Vega-Lite.
One challenge was ensuring consistent joins between statistical data and a geographic TopoJSON file. I addressed this by matching ISO numeric codes and validating joins programmatically in Python. This TopoJSON is also collected in the Google Colab notebook above.
Replication note: To reproduce the full process, open the Colab notebook and run. This will (1) query the World Bank API, (2) clean and harmonise the data and (3) output the CSV files used in the charts below.

Which countries are in cluster D?

This shows how many countries are in each cluster with k=4. Compared to k=3, cluster A is entirely unchanged, and most countries in cluster B and C remain in the same groups. The fourth cluster (cluster D) consists of only three countries - Ireland, Korea and Switzerland. Conclusion

By using standardised socio-economic indicators and clustering, this project shows that OECD countries can be more informatively understood as belonging to a small number of broad structural groups. At the same time, the analysis highlights the limits of a purely cluster-based perspective. While the results suggest that, under this indicator set, OECD countries are largely organised into three core structural patterns, additional clusters mainly capture a small number of structurally distinctive cases rather than fundamentally reshaping the overall structure. Overall, a combined perspective is informative: cluster-based analysis helps reveal underlying structural differences across countries, but careful attention to within-cluster diversity remains essential for meaningful cross-country comparison.
